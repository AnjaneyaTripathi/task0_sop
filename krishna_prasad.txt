git clone <url> to clone the repo into local system
git init - to initialize existing git repo
git branch <name>-to create a new branch
git checkout <branchname>-to access the branch
git add <filename>-adds the changes in the working directory in staging area
git commit-m"message"-save changes to the local repo
git status-displays state of the working directory
git push -uploads local repo content to remote repo
git pull -gets content from remote repo and updates local repo
